What is npm?

npm is the package manager for the node javaScript platform. It puts modules in place so that node can find them, and manage dependency conflicts intelligently. It is extremly configurable to support a wide variety of use cases.

What is Angular CLI?

The Angular CLI is a command line interface tool that you use to initialize, develop, and maintain angular applications directly from a command shell.

What is webpack in Angular?

Webpack is a popular module bundler, a tool for bundling application source code in convenient chunks and for loading that code from a server into a browser.

How does Angular achieve SPA?

In the SPA, the whole data is sent to the client from the server at the beginning. This results in a lesser load on the server and is cost-efficient. SPAs use AJAX and HTML5 to create a responsive web applications and most of the work happenes on the client side.

What are types of directives in Angular?
The three types of directives in Angular are attribute directives, structural directives, and components.
  • Attribute directives, are a number of built-in directives that we can add to our HTML elements that give them a dynamic behavior. In summary, an attribute directive changes the appearance or behavior of DOM element. EX: first we can create a directive by CLI command ng g d directiveName. After that, will import ElementRef, make a dependency injection to use it. So, now I can write for example el.nativeElement.style.backgroundColor = "yellow"; in the component in the html element tag will add the directive.
  • Structural directives, are directives which change the structure of the DOM by adding or removing elements. There are three built-in structural directives, ngIf, ngFor, and ngSwitch. These directives work by using the HTML5 tag.
  • Component directives, It forms the main class and is declared by @Component. It contains the details on component processing, and usage at run time.
What is the decorator in Angular?

Decorator marks a class as an Angular component and provides configuration metadata that determines how the component should be processed, instantiated, and used at runtime.

What types of forms in Angular?

There are two types of forms in Angular. Reactive forms and Template-driven forms. In short, reactive forms are more comprehensive than template-driven forms. More scalable, reusable, and testable. Use reactive forms.

How can we share data between components in Angular?
There are five ways to share data between components.
  • Parent to child component
  • Child to parent component
  • Sharing data between sibling components
  • Sharing data using ViewChild property
  • Sharing data between not related components

RxJS: is a Reactive Extensions for JavaScript, a library for reactive programming using observables that makes it easier to compose asynchrononus or callback-based code.

What is lazy loading in Angular and how to apply it?

Lazy loading is a technique in Angular that allows you to load JavaScript components asynchrononusly when a specific route is activated. It improves the speed of the application load time by splitting the application into several bundles. When the user navigates through the app, the bundles are loaded as required.
To create lazy loading in Angular we use the angular app routing module and main app module class to lazy load the angular component. You can effortlessly get the AppRoutingModule in the app-routing.module.ts file whereas AppModule class is found in the app.
